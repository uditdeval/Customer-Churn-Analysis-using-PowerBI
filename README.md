# Customer Churn Analysis using Power BI

## 📊 Project Overview
This project focuses on analyzing customer churn using Power BI.  
The dashboard helps identify churn patterns, customer behavior, and key factors that influence customer retention.

## 🎯 Objectives
- Analyze churn rate and customer segments
- Identify key factors affecting customer churn
- Visualize insights using interactive dashboards
- Support data-driven decision making

## 🛠 Tools & Technologies
- Power BI
- Power Query
- DAX
- Microsoft Excel (for data source)

## 📈 Dashboard Features
- Total Customers & Churn Rate
- Churn by Gender, Age Group, and Contract Type
- Monthly & Yearly churn trends
- Interactive slicers for better analysis

## 📂 File Information
- `.pbix` file contains complete data model, transformations, DAX measures, and visuals

## 🚀 How to Use
1. Download the `.pbix` file
2. Open it in **Power BI Desktop**
3. Refresh data if required
4. Explore the interactive dashboard

## 📌 Key Insights
- Certain customer segments show higher churn rates
- Contract type and usage patterns significantly affect churn
- Targeted retention strategies can reduce churn
